刻意

约 900 字大约 3 分钟

flower

2024-01-22

无意间再一次看到酷狗音乐播放界面，里面有滚动的歌曲的评论区，想起好久好久之前尝试看过一次这里的评论区，里面大概就是充满了各种各样神奇的故事，很多其实和歌曲完全没有关系，但是它能吸引你继续看下去，当时觉得很有意思，也就没管，感觉怪怪的。

后来可能是阅读文章，可能是与人交谈，我无意间发觉实际上歌曲的评论区就是 刻意 设计的一些个小故事，让你愿意去看，让你用尽量多的时间留在评论区。那时，我便莫名萌生出许多不喜欢， 之后每当看到循环滚动的评论区，即使我觉得很有意思，但我也一定不会点进去。

恰好今天在看完某一个知乎回答的之后，相似的发现了下面有推荐文章，标题叫 女人的直觉有多准，就那种几乎可以猜到会是怎么发展的故事，为了验证另一个猜想，点进去直接滚到底部，果然，是一片很长很长的，盐选会员文章。

也想起好久好久之前看过一篇这样的文章，当时是一个故事，大概是班上一个富哥带着 “我“ 一个成绩很好的穷小子鬼混，把 “我” 玩弄于鼓掌之间，被迫去恶搞了一个对 “我” 很好的老师，后面就因为不是盐选会员看不了了。

虽然现实生活中也会发生很魔幻的故事，但很明显这就是 刻意 编纂出来的小说，发现了其文章本质相同后，便索然无味了。

上次也想仔细思考这个问题，可惜因为我是鸽子，正如动态也想写但是没机会写完发出来，终于在今天再一次遇到契机。

我在思考，音乐的评论和盐选文章有做错什么吗，其实它们也没做错，只是我不喜欢 过度 刻意 而营造出来极度不自然的感受。

另一方面，你认真的写文章，写动态，发布之前再三的检查修正，这难道不是 刻意 为之的结果？

这里便应当要以自己的主观感受来评断 刻意 的程度与内容，

若是其过程重复的，产出的内容是低或无价值的 刻意，正如仅仅为了博人眼球的文章是我很不喜欢的。

个人而言，我尽力让自己变得不 刻意。我便不太喜欢如 无数遍的彩排，绝对精细的发言，天衣无缝的策划。必要时我会欣然接受，我会认真欣赏别人带来的完美表现并体谅其不易，但这本不是我所期望的。

我相信一句话 “人生没有彩排，每天都是现场直播”，绝对的 刻意 不一定会带来安全感或者预期的效果。

我不愿为此目的而做得绝对的 刻意 但愿为了内容，形式，将我的努力停留在 毫无准备 与 极度刻意 之间，

或许属于 有所准备，但尽力趋于 顺其自然。

你可能觉得我所形容的仅仅只是因为太懒了，连充分准备都不愿做到。

不，我会做到自己认可的认真准备，并自然的迎接 每一天。
