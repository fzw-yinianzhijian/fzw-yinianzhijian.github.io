临时鲜花

约 557 字大约 2 分钟

flower

2023-11-20

今天来随机写话！

感觉确实很久没有写鲜花了，最近都在发动态，但是感觉发动态好累的，每次考虑好久怎么说话才没那么奇怪，比如昨天其实还有校园寻宝活动想发动态（虽然好像被我们队打成了城市定向问题）

鲜花虽然也会考虑，但是实际上说话便要自由很多，算了，这篇鲜花就直接不 check 了，说什么是什么，看个乐呵就好。

感觉预科来对了，虽然没咋加训 OI 了，虽然期中考炸了，但是还是觉得预科很好玩，给了我截然不同的体验！

在考虑动态和鲜花的题材区别在哪里？实际上都是在写话，鲜花可能从角度而言，便是自己稍加限制的随意写话，动态属于是需要更为特别展示的一个角度，所以你对他的检查便会更多，相比之下，自己写的日记便随意很多了，虽然称为 "diary" 但是实际上是断断续续的，偶尔想起就写一写，但是总是咕掉。

或许写话也是一种支线，因为如果当前没有把脑海中想到的随机文字记录下来，那你便永远不可能知道这一刻的想法，和这一刻能写出来的文字是什么，之前其实也有记录好几个，可能加起来有 4/5 个鲜花素材了，可惜来不及写，而当再次拿起时，便只能依稀记得当时的一点点心境，可能也写不出来当时的感觉了。

虽然但是，写话总是有意思的，打字相比起直接说话是一个更为缓慢的过程，当然，如果用手写出来就又慢上一层，细细品味慢慢表达的感受，挺好。

信守诺言，完全没有检查上面的任何文字，写了就发，也不打算再做任何改动了，有什么不通顺，不合理，不正确，不积极的地方请大家见谅，看着玩吧。
